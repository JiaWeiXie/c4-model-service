# C4 Model Service

## Installtion

```bash
$ git clone https://github.com/JiaWeiXie/c4-model-service.git
$ cd c4-model-service
$ git submodule update --init --recursive
$ poetry install
```

## Run web app

```bash
$ cd c4-model-service
$ poetry shell

$ python main.py
```

> open `http://localhost:8088`
